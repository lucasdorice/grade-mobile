# grade-mobile

O Grade nasceu da necessidade de centralizar a rotina universitária. Mais do que um bloco de notas, é um diário de bordo pensado para estudantes de tecnologia. Ele permite importar grades de aulas, documentar conceitos, salvar blocos de código (JavaScript, HTML, CSS) passados em sala, e gerenciar prazos de trabalhos e lições de casa, evitando que entregas importantes se percam ao longo do semestre.
